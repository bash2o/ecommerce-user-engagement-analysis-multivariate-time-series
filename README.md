# E-commerce User Engagement / KPI Analysis (Multivariate & Time-Series)

## 📌 Project Overview
This project analyzes user behavior in an e-commerce platform using clickstream data.  
The goal is to understand the factors affecting user engagement (KPI) and identify patterns using multivariate and time-series analysis.

---

## 🎯 Objectives
- Analyze user engagement using session-based metrics
- Identify key factors affecting KPI performance
- Apply multivariate analysis to understand relationships between variables
- Perform time-series analysis to observe trends over time
- Generate insights to support data-driven decision making

---

## 📊 Dataset
- Source: E-shop clothing dataset (2008)
- Type: Clickstream data
- Duration: 5 months (April–August)
- Features include:
  - User session behavior
  - Product categories and models
  - Price and product attributes
  - Country and navigation data

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## 🧠 Methodology

### 1. Data Preparation
- Selected relevant features
- Handled dataset structure
- Created new variables:
  - `session_count` (engagement KPI)
  - `date` (time-series variable)

### 2. Feature Selection
Selected features:
- Country
- Product category and model
- Price and price level
- Colour
- Page (navigation)
- Month (time factor)

### 3. Analysis
- Multivariate analysis (factor-based)
- Time-series analysis (trend-based)
- Data visualization

---

## 📈 Key Insights
- User engagement varies across countries and product categories
- Certain products generate significantly higher interaction
- Pricing has a noticeable effect on user behavior
- Engagement is unevenly distributed among users (few highly active users)
- Engagement changes over time, indicating dynamic user behavior

---

## 💡 Conclusion
This project demonstrates how combining multivariate and time-series analysis can provide deeper insights into user behavior.  
The results can help improve KPI performance, optimize product strategies, and support business decision-making.
